# Project Description

Landing Page Project - This project contains a multi-section landing page. The navigation menu is automatically created based on the content of sections of the page. Clicking on a navigation item will scroll to the appropriate section of the page. This fixed bar will be hidden when user is no longer scrolling. While navigating through the page, the section that is active in the viewport will be highlighted from the other sections.
